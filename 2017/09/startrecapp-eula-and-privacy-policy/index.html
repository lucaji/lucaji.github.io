[The App uses un-encrypted peer-to-peer (P2P) data packets which relies on your Wifi-LAN (Local Area Network) to communicate with each other StartRecApp device (recorder). The App connects to Apple’s NTP time server (time.apple.com) to achieve a better time synchronisation between connected recorders when using timed controls.]{.s1}

[]{.s1}

[User Content. Each photograph taken and stored (MediaFile) belongs to one Take’s media file collection which in turn belongs to a Session’s takes collection.]{.s1}

[All recorded media is stored within the app’s Documents folder which is directly accessible using iTunes. Its structure mimics the above mentioned Session-Take-File model as you browse through it.]{.s1}
